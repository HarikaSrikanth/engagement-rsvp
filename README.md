# 💍 Engagement Invitation

A beautiful, responsive RSVP invitation website for an engagement ceremony.

## Features

- 🎥 Embedded video support
- 📱 Fully responsive design
- 📧 Automatic email notifications for RSVPs
- 🎨 Modern, elegant UI
- 📍 Event details and venue information
- 📊 RSVP tracking (number of guests, attendance status)
